mallzoom.js API Developer Guidev20180703

Overview

mallzoom.js is a jQuery image zoom slider plugin for e-commerce websites.

  • Only works properly on modern web browsers: IE8+, Chrome, Firefox, Opera, etc.
  • jQuery 1.7+ is required to be included in the page before the plugin. Download jQuery
  • Licensed under GNU GPL Version 3 (Open Source). License Document

Features

  • Simple Image Slider
  • Mouseover Image Zoom
  • HTML5/YouTube Video
  • Customizing Options

Usage

$('#sample').mallzoom(options);

Options

Param Type Default Description
ImgFit string 'fit'
outputPos string 'fit'
