mallzoom.js API Developer Guidev20180703

Overview

mallzoom.js is a jQuery image zoom slider plugin for e-commerce websites.

  • Only works properly on modern web browsers: IE8+, Chrome, Firefox, Opera, etc.
  • jQuery 1.7+ is required to be included in the page before the plugin. Download jQuery
  • Licensed under GNU General Public License, Version 3. License Document

Features

  • Simple Image Slider
  • Mouseover Image Zoom
  • HTML5/YouTube Video
  • Customizing Options

Downloads

  • Current release: 1.0 Package Download
  • Debug build: 1.1b Package Download

Usage

$('#sample').mallzoom(options);

Options

Param Type Default Description
ImgFit String 'fit' Type of image fitting.
['fit'] Fit the longer dimension of each image to its parent
['cover'] Crop each image if needed to cover its parent.
['stretch'] Stretch both dimension of each image to fill parent.
outputPos String 'right' Position of the zoomed output image.
['top', 'right', 'bottom', 'left']
thumbPos String 'bottom' Position of thumbnail images.
['top', 'right', 'bottom', 'left']
thumbCnt Integer 5 Quantity of thumbnails per slider page.
flagOver String '' Source URL of common overlay flag images.
txtOver String '' Common overlay text string.
zmScale Float 0.05
zmMax Integer 5
reverseZm Boolean false
noticeTxt String ''
noticeDur Integer 2
autoMovNav Boolean true
autoHidNav Boolean false
separToken String ''
loadingImg String ''
imgErr String ''
thumbErr String ''
vidErr String ''
vidBtn String ''
