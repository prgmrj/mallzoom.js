mallzoom.js API Developer Guide20180703

Overview

mallzoom.js is a jQuery image zoom slider plugin for e-commerce websites.

  • Only works properly on modern web browsers: IE8+, Chrome, Firefox, Opera, etc.
  • jQuery 1.7+ is required to be included in the page before the plugin. Download jQuery
  • Licensed under GNU General Public License, Version 3. License Document

Features

  • Simple Image Slider
  • Mouseover Image Zoom
  • HTML5/YouTube Video
  • Customizing Options

Downloads

  • Current release: 1.0 Download
  • Debug build(Beta): 1.1b Download

Usage

$('#object_id').mallzoom(options);
$('.object_class').mallzoom(options); // Multiple objects with the same options

Options

Parameter Type Default Description
ImgFit String 'fit' Type of the image-wrapper fitting.
['fit'] Fit the longer dimension of each image to its parent
['cover'] Crop each image if needed to cover its parent.
['stretch'] Stretch both dimension of each image to fill parent.
outputPos String 'right' Position of the zoomed output relative to the input image frame.
['top', 'right', 'bottom', 'left']
thumbPos String 'bottom' Position of thumbnail images relative to the input image frame.
['top', 'right', 'bottom', 'left']
thumbCnt Integer 5 Quantity of thumbnails per slider page.
flagOver String '' Source URL of common overlay flag images.
txtOver String '' Common overlay text string.
zmScale Float 0.05 Zooming scale per wheel scroll.
zmMax Integer 5 Maximum magnification.
reverseZm Boolean false Reverse wheel scroll direction for zooming.
noticeTxt String '' Overlay text over output image frame.
noticeDur Integer 2 Showing duration of output frame overlay text. (Seconds)
autoMovNav Boolean true Change slide on mouseover thumbnail images.
autoHidNav Boolean false Hide slide navigation arrows when every thumbnails are shown.
separToken String ';' Multiple data attributes separating token.
loadingImg String '' Source URL of the loading image.
imgErr String '' Source URL of the slide onerror image.
thumbErr String '' Source URL of the thumbnail onerror image.
vidErr String '' Source URL of the video onerror image.
vidBtn String '' Source URL of the play button image on video thumbnails.
